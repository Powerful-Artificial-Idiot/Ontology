# Manufacturing Knowledge Graph Explorer

A local frontend prototype for manufacturing knowledge engineering, including Route Explorer, Ontology Explorer, and Semantic Explorer.

## Features

- Route Explorer for product manufacturing process visualization
- Ontology Explorer for object type and relationship exploration
- Semantic Explorer for mapping business terms to ontology objects, system fields, source evidence, and AI context
- Production, Quality, Engineering, and Value Stream views
- Search, neighborhood highlighting, focus mode, collapsible sidebars, and detail panels
- Local TypeScript mock data with no backend dependency

## Tech Stack

- React
- TypeScript
- Vite
- React Flow
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

The development server is available at the URL printed by Vite.

## Build

```bash
npm run build
```

The production bundle is written to `dist/`.
