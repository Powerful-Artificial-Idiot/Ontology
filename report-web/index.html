需要管理层支持四项关键决策。

定范围确定一个试点产品或产线，以及一条主加工路线。
定 Owner指定业务负责人，组织生产、质量、工程与 IT 共同参与。
开数据允许读取有限范围的系统数据和文件样本。
定机制明确责任、审核、复盘与后续扩展边界。
